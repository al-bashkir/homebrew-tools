# Al-bashkir Tools

## How do I install these formulae?

`brew install al-bashkir/tools/<formula>`

Or `brew tap al-bashkir/tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "al-bashkir/tools"
brew "<formula>"
```

## Formulae

- **envio** ‚Äî modern, secure CLI for managing environment variables. `brew install al-bashkir/tools/envio`
- **ssh-tui** ‚Äî TUI for managing SSH connections. `brew install al-bashkir/tools/ssh-tui`
