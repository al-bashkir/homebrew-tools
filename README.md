# Al-bashkir Tools

## How do I install these formulae?

`brew install al-bashkir/tools/<formula>`

Or `brew tap al-bashkir/tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "al-bashkir/tools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
